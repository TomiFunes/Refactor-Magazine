# Refactor Magazine

A modern technology magazine and editorial portfolio focused on software engineering, artificial intelligence, programming, architecture, and digital innovation.

## 🚀 Live Demo

**Vercel Deployment:**
https://refactor-magazine.vercel.app

## 📖 Overview

Refactor Magazine is a modern editorial platform built to publish articles, insights, and technical content about software development and emerging technologies.

The project combines a premium dark-themed interface with a clean reading experience, providing a space for sharing knowledge, personal projects, technical articles, and industry insights.

## ✨ Features

- Modern responsive design
- Dark mode interface
- Bilingual support (English / Spanish)
- Authentication powered by Supabase
- Protected admin dashboard
- Article management system
- Featured articles
- Categories and organization
- SEO-friendly architecture
- Fast performance with Vite
- Deployment with Vercel
